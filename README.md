# rubik-vue-ant-admin
基于[rubik-cli](https://github.com/rubikjs/rubik-cli)和[Ant Design of Vue](https://pro.ant.design/index-cn)的后台系统框架

## 目的
减少UI相关的功能，只提供最基础和最核心的功能

## 功能列表

- 主题定制(ant完整变量全局访问)
- 环境配置
- 路由配置
- 权限配置
- 登录

## 依赖和参考

- axios
- vuex
- js-cookie
- ant-design-pro

## development
```
yarn start
```

## build
```
yarn build
```
